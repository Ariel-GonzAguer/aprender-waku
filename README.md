# Aprender Waku → Primera guía en español 

Esta es la primera guía en español para aprender el framework minimalista, React-First, basado en React Server Componentes: [Waku](https://wakujs.gg/).
Basada en la documentación oficial este proyecto nace del deseo de aprender Waku y la impresionante deficiencia de varios modelos de IA para crear una guía decente.

Esta guía está repleta de ejemplos de código, explicaciones detalladas y mejores prácticas para ayudarle a aprender Waku, sin tener que atravesar la barrera del idioma.

## Stack

- **Framework**: Waku 0.27.3 (React Server Components)
- **Runtime**: Node.js con Netlify Edge Functions
- **Lenguaje**: TypeScript 5.9
- **Estilos**: Tailwind CSS 4.1
- **Markdown**: React Markdown + Rehype + Remark GFM
- **Syntax Highlighting**: react-syntax-highlighter (Prism)

## Instalación

```bash
pnpm install
pnpm dev
```

Server corre en `http://localhost:3000`

## Scripts

```bash
pnpm dev          # Dev server
pnpm build        # Build de producción
pnpm start        # Ejecutar build
pnpm deploy:netlify # Deploy a Netlify
```
